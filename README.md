# FED2 JS2 Course Assignment – Social Media App

This project is my Course Assignment for FED2 JavaScript 2.  
The goal is to build the **front-end for a social media application** using HTML, CSS, and Vanilla JavaScript.

## Features (to be implemented)
- Register new users
- Login/logout functionality
- View feed (all posts)
- Create, edit, delete own posts
- View single post
- Search posts
- Profile page (my profile + others)
- Follow/unfollow users

## Tech
- HTML
- CSS
- JavaScript (ES6 modules)
- Noroff Social API v2
